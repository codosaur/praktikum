# Описание проекта

Построение модели для определения стоимости автомобилей с пробегом. Закачик делает упор на качество и скорость предсказаний.

# Инструментарий и навыки

- Catboost
- LightGBM
- NumPy
- Pandas
- Python
- Scikit-learn
- визуализация данных
- градиентный спуск
- предобработка данных
